AI-Ready Digital Curriculum Design

หลักสูตรต้นแบบที่ออกแบบจาก Skill Gap และความต้องการตลาดแรงงานยุค AI


Module 1

AI Literacy

พื้นฐาน AI, Generative AI, การใช้ AI Tools และข้อจำกัดของ AI

Module 2

Prompt Engineering

การเขียน Prompt เพื่อสรุป วิเคราะห์ วางแผน และสร้าง Workflow ด้วย AI

Module 3

Data Analytics

Excel, SQL, Power BI, Data Visualization และการตัดสินใจจากข้อมูล

Module 4

AI Ethics & Security

จริยธรรม AI, Privacy, Bias, Cyber Safety และการใช้ข้อมูลอย่างรับผิดชอบ

Module 5

Human + Career Skills

Communication, Problem Solving, Portfolio และการเตรียมตัวสมัครงาน

Module 6

Industry Project

โปรเจกต์จริงร่วมกับภาคอุตสาหกรรม เพื่อสร้าง Portfolio และประสบการณ์ทำงาน
